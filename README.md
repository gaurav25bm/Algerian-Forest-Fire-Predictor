# Algerian Forest Fire Predictor

This is a Machine Learning web application built with **Flask**. It uses a **Ridge Regression** model to predict the Fire Weather Index (FWI) based on Algerian Forest Fire data.

## Features
* Predicts FWI based on Temperature, Humidity, Wind Speed, and Rain.
* Web interface built with Flask and HTML.
* Model trained using Scikit-Learn.

## Deployment
This app is configured for deployment on **AWS Elastic Beanstalk**.
